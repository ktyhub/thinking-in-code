## Activiti是什么

Activiti是一个开源的业务流程管理平台，它帮助开发者设计和执行自动化工作流。简单来说，它就像一张智能地图，指导你的应用程序如何一步步完成复杂任务，比如审批申请或处理订单。通过直观的图形界面和强大的引擎，Activiti让流程管理变得轻松高效，无需深奥的代码知识即可上手。

## 入门示例

让我们走进一个真实场景：一家中型电商公司“速达商城”，他们每天面临数百个订单处理流程。过去，员工需要手动检查库存、审核支付、安排发货——这不仅耗时，还容易出错。引入Activiti后，他们设计了一个自动化工作流：当客户下单时，系统自动触发流程，先验证支付，再检查库存，最后生成发货单。如果库存不足，它会自动通知采购团队；一切顺利的话，订单在几分钟内完成处理。

开发示例方面，假设你使用Java和Spring Boot。首先，通过Maven添加Activiti依赖：

```xml
<dependency>
    <groupId>org.activiti</groupId>
    <artifactId>activiti-spring-boot-starter</artifactId>
    <version>7.20.0</version>
</dependency>
```

然后，定义一个简单的业务流程模型（BPMN文件），例如一个请假审批流程：员工提交申请，经理审批，系统自动更新记录。在代码中，你可以用几行命令启动流程：

```java
ProcessEngine processEngine = ProcessEngines.getDefaultProcessEngine();
RuntimeService runtimeService = processEngine.getRuntimeService();
runtimeService.startProcessInstanceByKey("leaveApprovalProcess");
```

这个示例不仅展示了Activiti的实用性，还让开发者能快速上手，将创意转化为现实。
