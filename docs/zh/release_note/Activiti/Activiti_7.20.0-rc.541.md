### **Activiti是什么**  
轻量级Java工作流引擎核武器。用**可视化流程图替代十万行if-else**，基于BPMN 2.0标准搭建企业级流程中枢，支持从请假审批到跨国供应链管理的任意场景。本质是**将业务流程转化为可执行代码的翻译器**。

### **入门示例：疫情物资调度系统**  
**业务痛点**：  
某慈善机构在疫情初期遭遇：捐赠申请积压2000+、人工派单失误率37%、物资追踪靠Excel截图  

**Activiti破局方案**：  
```java
// 1. 定义物资调度流程 (BPMN流程图)
[捐赠申请] -> (AI预审网关) -> {人工复核} -> [仓库配货] -> (物流竞标) -> [签收电子化]

// 2. 核心代码片段
Deployment deployment = repositoryService.createDeployment()
  .addClasspathResource("epidemic-supplies.bpmn20.xml")
  .deploy(); 

// 3. 启动紧急通道流程
runtimeService.startProcessInstanceByKey("emergencySupply", 
  variables.put("applicant","武汉协和医院")
           .put("priorityLevel",1));
```
**成效**：48小时上线，调度效率提升8倍，失误率归零。流程中嵌套的**AI预审网关**自动过滤重复申请，**物流竞标子流程**触发顺丰/京东实时报价——这正是Activiti的**多级流程嵌套**魔法。
