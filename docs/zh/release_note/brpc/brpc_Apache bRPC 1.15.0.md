# brpc Apache bRPC 1.15.0
## 为什么要使用brpc

想象一下，你正站在数字世界的十字路口，面对着数以千计的微服务像繁忙的都市交通一样交织。每个服务都在尖叫着需要高效通信，而传统的RPC框架却像老旧的电话交换机——缓慢、笨重、常常在关键时刻掉链子。这就是brpc登场的地方，它不是一个简单的工具，而是一场革命。brpc解决了现代分布式系统中最尖锐的矛盾：在追求极致性能的同时，如何保持代码的简洁与可维护性？当其他框架在复杂配置和低效吞吐量中挣扎时，brpc以惊人的效率撕裂了这些束缚，让开发者从繁琐的底层细节中解放出来，专注于创造价值。它就像一位无声的超级英雄，在后台默默处理每秒数百万的请求，却从不要求掌声——除非你开始对比使用前后的性能曲线，那时你会发现，没有brpc的世界，简直是一场数字噩梦。

## brpc是什么

brpc是Apache基金会旗下的一个开源RPC框架，全称为“baidu RPC”，但如今已远不止于此。
