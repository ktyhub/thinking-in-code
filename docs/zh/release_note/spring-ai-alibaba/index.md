## 📋 关于 Spring AI Alibaba

Spring AI Alibaba 是阿里巴巴开源的基于 Spring AI 构建的 Java AI 应用开发框架，提供与阿里云灵积模型服务以及通义大模型服务的无缝集成。它继承了 Spring AI 的设计理念，通过统一的 API 抽象，让开发者能够轻松切换不同的 AI 服务提供商。

## 🚀 主要特性

- **统一的 AI API**：提供一致的接口对接多种 AI 模型
- **通义大模型集成**：原生支持阿里云通义千问等大语言模型
- **Spring 生态集成**：无缝集成 Spring Boot、Spring Cloud
- **RAG 支持**：支持检索增强生成，集成向量数据库
- **函数调用**：支持 AI Agent 和工具调用能力
- **流式响应**：支持流式输出，提升交互体验
