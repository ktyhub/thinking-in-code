## 📋 关于 Spring Cloud Tencent

Spring Cloud Tencent 是腾讯开源的一站式微服务解决方案，基于 Spring Cloud 体系，集成了腾讯微服务平台 Polaris 的核心能力，提供服务治理、配置管理、服务路由等功能。

## 🚀 主要特性

- **服务治理**：集成 Polaris 服务发现与治理能力
- **配置管理**：动态配置中心支持
- **服务路由**：灵活的路由策略配置
- **限流熔断**：完善的服务保护机制
- **链路追踪**：全链路监控支持
- **Spring Cloud 兼容**：无缝集成 Spring Cloud 生态
