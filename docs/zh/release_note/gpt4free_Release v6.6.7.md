## 入门示例

凌晨三点的咖啡厅里，大学生李薇正在为她的社交创业项目发愁。预算只够买泡面，却需要智能客服系统。这时她发现了gpt4free：

```python
from g4f import Provider, ModelUtils

# 创建能理解中文语境的客服机器人
response = ModelUtils.create(
    model="gpt-4",
    provider=Provider.Yupp,
    messages=[{"role": "user", "content": "用户投诉快递延误该怎么安抚？"}]
)
print(response)
```

三行代码，她的项目活了。更震撼的是，她用这个工具帮视障朋友开发了语音读书助手——当AI遇见善意，技术便有了温度。开发者张明则用其构建代码审查工具，每次提交自动检测漏洞，团队效率提升200%，而他付出的代价仅仅是给项目点了个星。
