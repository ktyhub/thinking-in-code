### AnythingLLM是什么  
**您的企业级AI中枢**  
简单说，这是开源的「全栈智能工作台」。只需单行Docker命令，就能将任意大模型（开源/闭源）、任意文档（PDF/PPT/数据库）、任意工具链（搜索/爬虫/API）熔铸成定制化AI助手。它像乐高底座：插上LLM引擎是聊天机器人，接入公司文档是智能知识库，连接业务流程则化身自动化代理——所有数据100%留存于您的服务器。
