### tidb是什么

TiDB是一个开源的分布式数据库，旨在提供高可用性和水平扩展性。它兼容MySQL协议，支持SQL查询，适合在线事务处理（OLTP）和在线分析处理（OLAP）场景。TiDB的设计理念是将数据存储和计算分离，使得用户能够灵活地扩展资源，满足不断增长的业务需求。
