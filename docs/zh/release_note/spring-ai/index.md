## 📋 关于 Spring AI

Spring AI 是一个为 AI 工程提供抽象的应用框架，它为开发 AI 应用程序提供了可移植的 API 支持，包括与各种 AI 模型和服务的集成。Spring AI 继承了 Spring 框架的优点——依赖注入、自动配置、模块化设计，让 AI 集成变得像使用数据库或消息队列一样简单。

## 🚀 主要特性

- **统一的 API 抽象**：为各种 AI 模型和服务提供一致的接口
- **多模型支持**：支持 OpenAI、Azure AI、Amazon Bedrock 等多个 AI 服务提供商
- **Spring 生态集成**：与 Spring Boot 无缝集成，支持依赖注入和自动配置
- **RAG 支持**：内置检索增强生成（RAG）功能，支持向量数据库集成
- **函数调用**：支持 AI 模型的函数调用能力
- **流式响应**：支持流式输出，提升用户体验
