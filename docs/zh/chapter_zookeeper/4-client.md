
#  客户端的入门使用
##   命令行客户端
打开安装目录的bin目录输入命令

```bash
bin/zkCli.sh -server 127.0.0.1:2181
```
就可以进去zookeeper的脚本页面
可以输入help查看当前支持的命令的用法
