|                | 一致性             | 事务前后数据状态合法                               |
|                | 隔离性             | 并发事务互不干扰                                   |
|                | 持久性             | 提交后数据永久保存                                 |

|                | 链式事务           | 后续事务依赖前一个事务的提交结果                   |
|                | 嵌套事务           | 支持事务内嵌套子事务的结构                         |
|                | 分布式事务         | 跨节点/服务的事务，需协调一致                      |
