## 什么是MCP Server？

MCP (Model Context Protocol) Server是一种专门设计的服务器，用于处理大型语言模型(LLM)与外部上下文的交互。它允许模型访问、理解和利用外部知识库，从而增强模型的回答能力和准确性。
